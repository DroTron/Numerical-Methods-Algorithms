# luFactor.m
This MatLab R2018b program creates an upper triangular matrix and a
lower triangular matrix based on an inputed matrix.

# I/O:
1. Inputs
   * A - an n x m square matrix
2. Outputs
   * L - lower triangular matrix
   * U - upper triangular matrix
   * P - pivot matrix
3. Help
   * [L,U,P] = luFactor(A);
